# Jayanthi Lamoni - Personal Portfolio

Welcome to the source code of my personal portfolio website! This repository powers my GitHub Pages site hosted at [jayanthilamoni.github.io](https://jayanthilamoni.github.io/).

## 🎨 Overview

This is a modern, responsive, and minimalistic personal portfolio designed to showcase my experience, projects, and articles. It features a clean **Material Design 3** aesthetic with full support for both **Light and Dark mode** depending on your system preferences or via a manual toggle switch.

## ✨ Features

- **Materialistic Design**: Beautiful, responsive layout featuring card elevations, smooth shadows, and polished typography.
- **Dark/Light Mode**: Seamlessly switch between themes. The website remembers your preference via `localStorage` and defaults to your system preference.
- **Experience Timeline**: A clean, chronological timeline of my professional roles and responsibilities.
- **Projects & Articles Showcase**: Dedicated sections to highlight key open-source work and technical writing.
- **No Build Steps**: Pure HTML, CSS, and Vanilla JavaScript. Fast, lightweight, and easy to maintain.

## 🛠️ Technology Stack

- **HTML5**: Semantic and accessible structure.
- **CSS3**: Vanilla CSS with modern features like CSS Variables (Custom Properties) and Flexbox/Grid for layout.
- **Vanilla JavaScript**: Lightweight DOM manipulation for smooth scrolling and theme toggling.
- **Google Fonts**: Utilizing `Roboto` and `Product Sans` for crisp typography.
- **FontAwesome**: Used for scalable vector icons.

## 🚀 Deployment

This site is automatically deployed using **GitHub Pages**. Any changes pushed to the `main` branch will be reflected live on the website within minutes.
